# 7.1 스태시
스태시를 통하여 브랜치간 이동시에 작업사항이 남아있을경우 </br> 
커밋을 하지 않더라도, 임시저장해둔 채로 이동할 수 있다.

## 스태시 생성
git stash

## 스태시 삭제
git stash drop

## 스태시 목록 열람
git stash list

## 스태시 불러오기
git stash pop

## 스태시 복원으로 충돌
복원중 충돌이 생기면 스태시내용은 삭제되지 않는다. </br>
이때는 수동으로 수정하고 스태시를 삭제해줘야된다.
